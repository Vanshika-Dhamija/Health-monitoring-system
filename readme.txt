Compiling Instructions-
The code consists of two part , One part is responsible for calculating SpO2 and Heart Rate and other part is responsible for calculating GSR Value of human body and Human Resistance.

For part One-
1) Upload the code in Node MCU.
2) Put the Finger on the sensor once the code is uploaded.
3) Open Serial Moniter from tools to see the values.
4) Open Cayanne Dashboard to see the Values in IOT Cloud.

For part Two-
1) Upload the code in Node MCU.
2) Put the Finger inside the cover of the sensor once the code is uploaded.
3) Open Serial Moniter from tools to see the values.
4) Open Cayanne Dashboard to see the Values in IOT Cloud.
